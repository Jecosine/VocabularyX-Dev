# VocabularyX

VacabularyX is a simple software to memorize word, providing features:

## Words Management

Providing application interface to manage words. 

- Words CRUD
- User friendly UI
- Use Chapters and Prefix Tags to group words

## Words Memorize

- Hide `Spellings` | `Meaning` | `Part of Speech`


## Words Contest

- Similar words
- Similar meanings
- Similar Prefix
