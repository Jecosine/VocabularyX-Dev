# VocabularyX [绝赞开发中]

## 前言

**目前软件还在开发中**

## 介绍

VocabularyX **将**是一款具备以下功能的英语单词辅助背诵软件

- 自定义词库
- 通过单词库之间进行集合运算产生新的笔记本
- 自动生成单词本的背诵计划任务
- 自动生成单词的练习
  - 句子填空
  - 单词听写与默写
  - 挑战模式（可多人？）
- 统计学习记录
- 可以通过呼出命令箱完成绝大部分的操作内容（狂喜
- 您也许喜欢的界面布局

## 支持的平台

- Linux发行版 
- Windows 7 以上
- MacOS
- 可部署版本 支持大部分主流的浏览器
